# CAN_boot
Загрузчик для ATMega328P, позволяющий осуществить прошивку по CAN

Основан на стандартном загрузчике для ATMega328P - Optiboot

# Как это использовать (в общих чертах)

1) Залить стандартным образом загрузчик на микроконтроллер
2) Подключить его к прошивающему МК
3) Через него передавать прошивку
